# ProjetoBAC teste

O arquivo deploy.sh deve ser rodado no servidor na pasta /opt/deploy_bac/deploy_bac/ <br>
Esse arquivo verifica se tem alterações na branch develop e sobe o strapi e atualiza o banco de dados. <br>

````
bash ./deploy.sh
````
